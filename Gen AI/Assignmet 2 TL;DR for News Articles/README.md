#  TL;DR AI News Summarizer

A simple and elegant Streamlit web app that uses Generative AI to convert long news articles into short, readable summaries.

---

##  Features

- Summarizes long news articles into a 3-sentence TL;DR
- Powered by Hugging Face Transformers
- Uses the DistilBART CNN summarization model
- Clean, modern user interface
- Live word count display
- Fast performance with cached model loading

---

##  Tech Stack

- Frontend: Streamlit
- Language: Python
- Model: sshleifer/distilbart-cnn-12-6
- Library: Hugging Face Transformers
